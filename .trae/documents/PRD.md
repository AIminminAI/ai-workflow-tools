# AI Recipe Shop - 产品需求文档（PRD）

## 1. 产品概述

**AI Recipe Shop - 拒绝理论，直接拿走搞钱工作流** 是一款面向自媒体、电商、独立开发者的单页应用（SPA），通过智能筛选匹配高价值 AI 工作流配方，并以"核心壁垒付费解锁"模式实现商业化变现。

- **核心目的**：解决用户"知道 AI 能赚钱但不知道具体怎么组合使用"的痛点，提供开箱即用的 AI 工具链配方
- **目标用户**：小红书博主、跨境电商卖家、自媒体剪辑师、职场打工人
- **商业价值**：通过 1.9 元微付费解锁核心 Prompt 模板与隐藏工具链，实现知识付费变现

## 2. 核心功能

### 2.1 用户角色

本产品为单角色（访客用户），无需注册登录，直接浏览与解锁。

### 2.2 功能模块

1. **单页应用主页**：顶部横幅 + 智能筛选区 + 配方卡片瀑布流

### 2.3 页面详情

| 页面名称 | 模块名称 | 功能描述 |
|---------|---------|---------|
| 主页 | 顶部 Header 与横幅 | 产品 Title、副标题、品牌氛围营造 |
| 主页 | 智能筛选区 | 身份选择（4 种）、预算倾向（3 种）、目标产出（3 种）三轴联动筛选 |
| 主页 | 配方卡片展示区 | 根据筛选动态展示 2-3 个匹配工作流卡片 |
| 主页 | 卡片-工具链 Badge | 优雅展示 [工具A] ➔ [工具B] ➔ [工具C] 组合拳 |
| 主页 | 卡片-步骤拆解 | 1/2/3 步极简文字说明 |
| 主页 | 卡片-核心壁垒区 | 默认模糊遮罩，付费解锁后展示 Prompt 模板与隐藏工具链接 |
| 主页 | 付费解锁 Modal | 展示付款二维码占位图 + "我已支付"测试按钮 |
| 主页 | 一键复制提示词 | 解锁后 Prompt 右侧复制按钮，调用 navigator.clipboard |

## 3. 核心流程

用户进入页面 → 选择身份/预算/目标 → 系统动态匹配配方卡片 → 浏览卡片公开内容（名称/工具链/步骤）→ 点击"立即解锁" → 弹出付费 Modal → 点击"我已支付" → 当前卡片核心壁垒区解除模糊 → 查看完整 Prompt 并可一键复制。

## 4. 用户界面设计

### 4.1 设计风格

- **整体风格**：2026 年科技感暗黑极简风（参考 Linear、Vercel）
- **主色调**：深色背景 `slate-950`，卡片背景 `slate-900/50`，边框 `border-slate-800`
- **高亮色**：电光蓝 `blue-500`、极客绿 `emerald-500`
- **按钮风格**：圆角胶囊式，带 Hover 缩放与发光过渡
- **字体**：Geist Sans（Next.js 内置）作为正文，Geist Mono 用于 Prompt 代码块
- **布局**：桌面端左右分栏（左筛选右卡片），移动端上下堆叠
- **图标**：Lucide React 图标库
- **动效**：卡片 Hover 上浮、Modal 淡入缩放、模糊解除过渡

### 4.2 页面设计概览

| 页面名称 | 模块名称 | UI 元素 |
|---------|---------|---------|
| 主页 | Header 横幅 | 渐变背景、大标题、副标题、装饰光晕 |
| 主页 | 筛选区 | Tabs/Radio 身份选择、Button 预算组、Select 目标下拉 |
| 主页 | 卡片区 | 网格布局卡片、Badge 工具链、有序列表步骤、模糊遮罩 |
| 主页 | Modal | 居中弹窗、二维码占位、支付按钮、关闭按钮 |

### 4.3 响应式

- **桌面优先**：`lg` 断点以上左右分栏布局
- **移动适配**：`md` 以下上下堆叠，筛选区可折叠
- **触摸优化**：按钮最小点击区域 44px，间距充足

## 5. 数据结构

配方数据为 JSON 数组，字段定义：

| 字段 | 类型 | 说明 |
|------|------|------|
| id | number | 唯一标识 |
| role | string | 适用身份 |
| budget | string | 预算倾向 |
| target | string | 目标产出 |
| title | string | 配方名称 |
| tools | string[] | 工具链数组 |
| steps | string[] | 步骤数组 |
| hiddenPrompt | string | 解锁后展示的 Prompt |
| isLocked | boolean | 是否默认锁定 |

初始提供 3 条高质量数据，支持后续扩展。
